## 1. 쇼핑 쇼츠 자동화 프로젝트: MVP 범위, 수익화 전략 및 KPI 확정

타깃 청중(30~50대 한국인 여성)에 초점을 맞춘 '쇼핑 쇼츠 자동화' 프로젝트의 MVP 정의, 초기 수익화 모델, 그리고 핵심 성과 지표(KPI)를 아래와 같이 확정합니다.

### 1.1. MVP 범위 정의 (Minimum Viable Product)

**핵심 가설:** 타깃 청중은 '시간 절약'과 '트렌드에 맞는 콘텐츠 제작의 용이성'에 가장 높은 가치를 둔다.

| 항목 | 상세 내용 | 근거 및 설명 |
| :--- | :--- | :--- |
| **제품 형태** | **자동화된 쇼츠 콘텐츠 기획 및 초안 생성 서비스 (SaaS 형태)** | 단순 툴 제공보다 '결과물'에 대한 가치를 높여 구독 전환율을 극대화. |
| **MVP 기능 범위** | 1. **트렌드 분석 모듈:** 특정 카테고리(예: 패션, 홈데코)의 인기 쇼츠 트렌드 실시간 분석. | 타깃 청중의 즉각적인 관심사 반영. 경쟁 우위 확보. |
| | 2. **스토리보드 자동 생성:** 분석된 트렌드를 기반으로 촬영 구도 및 자막 초안 자동 제안. | 제작 시간 대폭 단축 (자동화 핵심). |
| | 3. **템플릿 기반 편집 가이드:** 타깃 청중 선호 스타일의 편집 템플릿 제공. | '자가학습' 채널의 정체성(스타일)을 녹여냄. |
| **제외 범위 (Phase 2)** | 실제 영상 제작 대행, 라이브 커머스 연동, 상세 판매 플랫폼 직접 연결 기능. | MVP 단계에서는 자동화 및 기획에 집중하여 기술적 리스크 최소화. |

### 1.2. 초기 수익화 전략 (Pricing Strategy)

**전략:** '시간 절약'과 '전문성 확보'에 대한 가치를 기반으로 **Tiered Subscription Model (단계별 구독 모델)**을 채택합니다.

| 티어 (Tier) | 가격 (월) | 주요 기능 | 목표 고객 |
| :--- | :--- | :--- | :--- |
| **Starter (입문자)** | 19,000원 | 트렌드 분석 리포트 및 스토리보드 초안 생성 (제한된 콘텐츠). | 취미로 시작하는 사용자. |
| **Pro (핵심 고객)** | 49,000원 | Starter 기능 + 맞춤형 템플릿 라이브러리 + 고급 편집 가이드 제공. | 꾸준히 콘텐츠를 생산하려는 주부/소규모 셀러. **(주력 타깃)** |
| **Premium (전문가)** | 99,000원 | Pro 기능 + 커뮤니티 접근 권한 + 월별 트렌드 심층 리포트 및 전문가 피드백 연동. | 콘텐츠를 비즈니스화하려는 사용자. |

**가격 근거:**
*   30~50대 여성은 높은 품질과 명확한 가이드에 기꺼이 비용을 지불하는 경향이 있음.
*   Pro 티어(49,000원)는 콘텐츠 제작의 '고통(Pain Point)'을 해결해주는 수준으로 설정하여 충분한 ROI를 제공할 것으로 판단.

### 1.3. 핵심 KPI 설계 (Key Performance Indicators)

수익화 모델 검증 및 장기 목표 달성을 위해 다음 세 가지 KPI를 최우선으로 운영합니다.

| KPI | 측정 지표 | 목표치 (3개월) | 목표 연관성 |
| :--- | :--- | :--- | :--- |
| **1. 전환율 (Conversion Rate)** | 무료 체험 → 유료 구독 전환율 | 5% 이상 | 수익화 모델의 직접적인 성공 여부 측정. |
| **2. 고객 생애 가치 (LTV)** | 평균 구독 기간 대비 총 매출액 | 월 구독료 대비 3개월 유지율 60% 달성 | 장기적인 수익 안정성 확보. |
| **3. 활성 사용자 참여도 (Engagement)** | 주간 유료 사용자당 콘텐츠 생성 시도 횟수 | 주당 평균 3회 이상 콘텐츠 초안 생성 | 제품의 사용 편의성과 가치 검증. |
